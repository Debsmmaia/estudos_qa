# 🧪 Laboratório de QA: Automação com Selenium & Documentação de Testes

Este repositório foi criado como parte do meu plano de estudos focado em Quality Assurance (QA) e Automação de Testes. O objetivo principal é consolidar conhecimentos práticos em testes de software, utilizando como objeto de estudo a plataforma Sauce Demo - Shopify.

## Métodologias usadas

1. Criação de plano de testes
2. Criação de casos de uso
3. Criação de casos de teste
4. Testes manuais
5. Registros de bugs no sistema
6. Automação de testes - Selenium WebDriver
