Este documento descreve o plano de testes da plataforma web Sauce Demo, tendo como foco a validação das funcionalidades principais do site

# Objetivo
• Validar o funcionamento da aplicação

• Identificar bugs e inconsistências

• Garantir que o fluxo principal do usuário funcione corretamente

• Servir como base para automação com Selenium

# Escopo

##Dentro do escopo:

• Criação de um novo usuário

• Login de usuário

• Navegação após login

• Adição de itens ao carrinho

• Remoção de itens do carrinho

• Finalização de compra (checkout)

##Fora do escopo:

• Teste de segurança

• Integrações externas

• Testes de performance

# Estratégia de Teste

Os testes serão realizados em duas etapas, primeiramente com os testes manuais para validação inicial e em segundo momento feito com automação utilizando Selenium

# 🛠️ Ferramentas

• Linguagem: Python

• Automação: Selenium

• Versionamento: GitHub
