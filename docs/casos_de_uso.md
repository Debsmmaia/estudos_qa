# 📋 Casos de Uso - Sauce Demo

## UC-01 - Cadastro de usuário

**Ator:** Usuário

**Descrição:**  
Permite que um novo usuário crie uma conta na plataforma.

**Fluxo principal:**
1. Usuário acessa a página inicial do site
2. Clica no botão de cadastro
3. Sistema redireciona para a página de cadastro
4. Usuário preenche os campos obrigatórios (nome, sobrenome, e-mail e senha)
5. Usuário clica no botão de finalizar cadastro
6. Sistema cria a conta do usuário
7. Usuário é redirecionado para seu perfil

---

## UC-02 - Login de usuário

**Ator:** Usuário

**Descrição:**  
Permite que o usuário acesse sua conta na plataforma.

**Fluxo principal:**
1. Usuário acessa a página inicial
2. Clica no botão de login
3. Sistema redireciona para a página de login
4. Usuário insere e-mail e senha
5. Clica no botão de login
6. Sistema valida os dados
7. Usuário é redirecionado para sua conta

---

## UC-03 - Adicionar produto ao carrinho

**Ator:** Usuário 

**Descrição:**  
Permite que o usuário adicione produtos ao carrinho de compras.

**Fluxo principal:**
1. Usuário acessa a página principal
2. Visualiza os produtos disponíveis
3. Seleciona um produto
4. Clica em "Adicionar ao carrinho"
5. Produto é adicionado ao carrinho

---

## UC-04 - Finalizar compra

**Ator:** Usuário 

**Descrição:**  
Permite que o usuário finalize a compra dos produtos selecionados.

**Fluxo principal:**
1. Usuário acessa o carrinho
2. Verifica os produtos adicionados
3. Clica no botão de finalizar compra
4. Insere os dados de entrega
5. Insere os dados de pagamento
6. Confirma a compra
7. Sistema finaliza o pedido
