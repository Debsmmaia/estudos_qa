## UC-01 - Login de usuário

**Ator:** Usuário

**Descrição:**  
Permite que um usuário acesse sua conta no sistema.

**Fluxo principal:**
1. Usuário acessa a página de login
2. Insere e-mail e senha
3. Clica no botão de login
4. Sistema valida os dados
5. Usuário é redirecionado para sua conta
